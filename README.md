# admin

## 개요

이 저장소의 목적은 풀씨 임원진의 할 일을 추적하고 기록해 임원진 활동의 연속성을 확보하는데 있습니다.  
제대로 문서화되거나 아카이브되기 힘든 실제 활동들을 남겨두어 쓸데없는 소통 비용을 줄입니다.
